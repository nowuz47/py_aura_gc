AuraGC

Python 3.14를 위한 인프라 인지형 가비지 컬렉션 가버너. 메모리 부족(OOM)으로 인한 컨테이너 재시작을 선제적으로 방지합니다.

Github 원본보기 최신 릴리즈 v0.1.0

Proactive Defense System

기존의 수동적인 GC를 넘어, 시스템 압력을 실시간으로 모니터링하여 대응합니다.

⚡

Native PSI Sensing

Linux Kernel의 Pressure Stall Information(PSI)을 C로 직접 읽어 10ms 미만의 지연시간으로 급격한 부하를 감지합니다.

🛡️

Emergency Brake

300MB 안전 한계점을 설정하여 컨테이너가 512MB 하드 리밋에 도달하기 전 선제적으로 Aggressive GC를 수행합니다.

🔒

Sawtooth Stability

생존한 객체를 즉시 Immortal Branding(Freeze) 처리하여 대규모 누수 상황에서도 메모리 안정성을 유지합니다.

The Sawtooth Result

Baseline(Red)은 반복적으로 OOM Crash가 발생하지만, AuraGC(Green)는 안정적인 톱니바퀴 곡선을 그립니다.
